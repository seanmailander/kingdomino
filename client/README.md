# Kingdomino Client

This app now uses Vite.

## Available Scripts

In the `client` directory:

### `npm run start`

Starts the Vite development server.

### `npm run start-host`

Starts the dev server on `kingdomino.local:80`.

### `npm run start-secure`

Starts the dev server on `https://[::]:443` using:

- `SSL_CRT_FILE`
- `SSL_KEY_FILE`

### `npm run test`

Runs tests with Vitest.

### `npm run build`

Builds for production into `build/`.

### `npm run preview`

Serves the production build locally.
